# 📰 TruthScan - Fake News Detection System

TruthScan is a Machine Learning-based web application that detects whether a news article is REAL or FAKE using Natural Language Processing (NLP). The application is built using Python, Flask, Scikit-learn, and TF-IDF Vectorization. It is trained on a large labeled dataset of real and fake news articles and predicts the authenticity of complete news articles through a simple web interface.

## ✨ Features

- Detects REAL and FAKE news articles
- Natural Language Processing (NLP)
- TF-IDF Vectorization
- Multiple ML model comparison
- Confidence score for predictions
- Interactive Flask web interface
- Prediction history
- Responsive UI
